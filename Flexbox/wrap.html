Flex Wrap

Wrap signfica "empacotamento" e é uma propriedada colocada apenas no container pai.

Antes de entrar nos valores, o que é esse empacotamento? Quando acontece?

Nesse exemplo, existem os 3 quadrados e uma enorme espaço em preto sobrando. O empacotamento, no caso de flexbox, vai ocorrer quando todo esse espaço preto acabar e, nesse encurtamento, começar a tomar espaço dos itens (ou seja, das caixas brancas). Quando chegar nesse ponto, os itens vão encolher sua largura (e altura também, se for o caso) para caber dentro desse espaço.

Porém, isso vai depender do conteúdo. Alguns (como textos muito grandes, imagens e outros desse tipo) vão quebrar em vez de encolher.

Nowrap

Dito isso, o nowrap é um valor padrão, ou seja, mesmo que eu não faça definição alguma será esse valor definido pelo próprio navegador. Ele instrui justamente isso: para não quebrar o conteúdo (o item) se o tamanho da tela reduzir. Em vez disso, é para encolher. Mais do que simplesmente não encolher, essa função visa deixar os elementos na disposição que coloquei (por exemplo, caso eu queira manter 3 itens na mesma linha).

Eixos E Wrap

Essa é uma coisa para ter em mente. Quando eu defino wrap, eu estou instruindo a quebrar o conteúdo. Mas para qual direção ele vai? Se eu tenho 3 itens na horizontal e no espaço só cabem 2, para onde vai o terceiro? Para cima ou para baixo?

Ele vai seguir a direção do eixo transversal. Sempre que ocorrer uma quebra, se trata do eixo transversal.

wrap-reverse

O wrap-reverse diz que, quando o espaço acabar e a quebra for acontecer, o item deve ir em direção oposta ao eixo transversal, ou seja, ao eixo vertical.

Porém, tem um detalhe importante. Isso se aplica tanto a elementos que estejam como row quanto como column. A diferença é que se estiver como linha, ele vai para a linha de baixo (ou de cima, se for wrap-reverse), e se for columns vai para a coluna ao lado esquerdo (se for normal) ou direito (Se for reverse).

No caso abaixo, o eixo está no modo padrão, de cima para baixo. Coloquei o mesmo quadrado, com os mesmos itens e com as mesmas cores na mesma sequência que o de cima, porém neste eu coloquei wrap-reverse.

Flex-flow

Flex-flow é um shorthando: primeiro eu delaco a direção e depois o wrap:

  • flex-flow: row wrap
  • flex-flow: column nowarp
  • flex-flow: column-reverse wrap-reverse
