Flexbox e GridLayout

Essas 2 novas tecnologias vieram para o CSS em sua atualização de 2017, visando resolver problemas de responsividade. As caixas, presentes nas versões anteriores de CSS e HTML, eram rigidas e não ofereciam alterações flexiveis. As 2 novas tecnologias vieram para resolver esse problema, tornando as caixas que compõem um site mais maleáveis.

Qual A Utilidade do Flexbox?

Como dito acima, o flexbox e o GridLayout (assunto do próximo capitulo) vieram para resolver um problema. No caso do flexbox, que problema ele resolve? O flexbox não tem utilidade se eu precisar de medidas exatas e inflexiveis, porém, caso eu precisa reposicionar as caixas, ele é tem muita utilidade. Em outras palavras, seu eu precisar de flexibilidade na construção da página, o flexbox será muito útil.

IMPORTANTE DETALHE

Display é uma propriedade geral. O que isso quer dizer? Até agora eu vi só display block ou display inline. Talvez dê a entender que display é intrinseco a block ou inline, ou seja, são a mesma coisa. mas isso não é verdade. Display, como acabei de dizer, é uma propriedade geral que não está limitado ou intrinseco a um valor ou outro. Dito isso, existem os seguintes valores para display:

  • Block: faz a caixa ocupar toda a largura da tela e cria quebra de linhas;
  • Inline: ocupa apenas a largura do elemento, ocupando a mesma linha;
  • None: faz com que o elemento não seja exibido;
  • Flex: ativa a tecnologia flexbox (permitindo usar seus recursos, se não usar display flex, não dá para usar, justamente porque não foi ativada);
  • Grid: ativa a tecnologia GridLayout, permitindo usar seus recursos.

Por definição, a propriedade display diz qual tipo de caixa o elemento terá e, com isso, como ele se comportará na pagina.

Container

Toda vez que eu for trabalhar com flexbox, que vai receber o display flex é o pai. E quem é esse "pai"? É o container, ou seja, a caixa onde todas as outras caixas que vou trabalhar com a intenção de serem flexiveis (todos os elementos) vão estar dentro. Os itens dentro do container, por sua vez, não vão receber o display flex, mas outros valores.
